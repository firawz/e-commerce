# E-Commerce-cms
List of available endpoints:
- `POST /login`
- `POST /loginUser`
- `POST /register`
- `GET /products`
- `GET /products/:id`
- `POST /products`
- `PUT /products/:id`
- `DELETE /products/:id`
- `GET /cart`
- `PUT /cart`
- `POST /cart/product/:id`
- `DELETE /cart/:id`
- `PATCH /cart/:id/quantity`
- `PATCH /cart/:id/status`
